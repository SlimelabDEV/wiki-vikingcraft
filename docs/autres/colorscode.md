# 🌈 Les codes couleurs Minecraft

Les codes couleurs Minecraft permettent de modifier la couleur et le style du texte affiché dans le jeu.

Ils sont très utilisés sur les serveurs Minecraft pour personnaliser les grades, les messages, les menus, les panneaux, les commandes et les plugins.

Dans ce guide, découvrez tous les codes couleurs Minecraft ainsi que leur utilisation.

## 🎨 Liste des codes couleurs Minecraft

Minecraft utilise le symbole `§` suivi d'un caractère spécifique pour définir une couleur.

| Couleur      | Code |
| ------------ | ---- |
| Noir         | `§0` |
| Bleu foncé   | `§1` |
| Vert foncé   | `§2` |
| Cyan foncé   | `§3` |
| Rouge foncé  | `§4` |
| Violet foncé | `§5` |
| Or           | `§6` |
| Gris clair   | `§7` |
| Gris foncé   | `§8` |
| Bleu         | `§9` |
| Vert         | `§a` |
| Cyan         | `§b` |
| Rouge        | `§c` |
| Rose         | `§d` |
| Jaune        | `§e` |
| Blanc        | `§f` |

## ✨ Codes de mise en forme Minecraft

En plus des couleurs, Minecraft propose plusieurs styles de texte.

| Style         | Code |
| ------------- | ---- |
| Obfusqué      | `§k` |
| Gras          | `§l` |
| Barré         | `§m` |
| Souligné      | `§n` |
| Italique      | `§o` |
| Réinitialiser | `§r` |

## 💬 Exemple d'utilisation

Voici quelques exemples :

```text
§aBienvenue sur le serveur !
```

Affichera un texte vert.

```text
§cAttention !
```

Affichera un texte rouge.

```text
§6Boutique Minecraft
```

Affichera un texte doré.

```text
§l§eGrade VIP
```

Affichera un texte jaune en gras.

## ⚙️ Utiliser les couleurs dans les plugins Minecraft

La plupart des plugins Minecraft utilisent le symbole `&` à la place du symbole `§`.

Exemple :

```yaml
message: "&aBienvenue sur le serveur !"
```

Le plugin convertira automatiquement :

```text
&a
```

en :

```text
§a
```

Cette méthode est utilisée sur de nombreux plugins comme :

- LuckPerms
- EssentialsX
- DeluxeMenus
- ItemsAdder
- BossShopPro
- TAB
- CMI

## 🌈 Couleurs HEX Minecraft

Depuis les versions récentes de Minecraft, il est possible d'utiliser des couleurs HEX.

Exemple :

```text
#55FF55
```

ou selon certains plugins :

```text
&#55FF55
```

Les couleurs HEX permettent d'obtenir des milliers de nuances différentes.

### Exemple de couleur personnalisée

```text
&#FF9900Boutique
```

Affichera un texte orange personnalisé.

## 🏷️ Utiliser les couleurs pour les grades Minecraft

Les serveurs Minecraft utilisent souvent les couleurs pour distinguer les grades.

Exemples :

```text
&7[Joueur]
```

```text
&a[VIP]
```

```text
&b[VIP+]
```

```text
&6[Héros]
```

```text
&c[Admin]
```

Cela permet aux joueurs d'identifier rapidement les différents rôles du serveur.

## 🛠️ Résoudre les problèmes courants

### Les couleurs ne fonctionnent pas

Vérifiez que :

- Le plugin supporte les codes couleurs.
- Vous utilisez le bon symbole (`&` ou `§`).
- Les permissions sont correctement configurées.

### Les couleurs HEX ne s'affichent pas

Assurez-vous :

- D'utiliser une version récente de Minecraft.
- Que le plugin supporte les couleurs HEX.
- D'utiliser le bon format demandé par le plugin.

### Le texte reste blanc

Essayez de :

- Vérifier la configuration du plugin.
- Relancer le serveur.
- Tester avec un code couleur simple comme `&a`.
