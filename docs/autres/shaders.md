# ✨ Installer des shaders sur Minecraft

Les shaders permettent d'améliorer considérablement les graphismes de Minecraft en ajoutant des effets de lumière réalistes, des ombres dynamiques, des reflets sur l'eau, des nuages volumétriques et bien d'autres améliorations visuelles.

Dans ce guide, découvrez comment installer des shaders sur Minecraft Java Edition rapidement et facilement.

## 📋 Prérequis

Avant de commencer, vous devez disposer :

- D'une version de Minecraft Java Edition
- D'un ordinateur suffisamment performant
- D'un pack de shaders compatible
- D'OptiFine ou d'Iris Shaders

## 📥 Télécharger un pack de shaders Minecraft

De nombreux packs de shaders sont disponibles gratuitement sur Internet.

Avant de télécharger un shader, vérifiez :

- Sa compatibilité avec votre version de Minecraft
- Les performances recommandées
- Les éventuels prérequis indiqués par l'auteur

Une fois téléchargé, conservez le fichier ZIP sans le décompresser.

## ⚡ Installer des shaders avec OptiFine

OptiFine est l'une des solutions les plus populaires pour utiliser des shaders sur Minecraft.

### Installer OptiFine

- 1. Téléchargez la version compatible avec votre version de Minecraft.
- 2. Lancez le fichier d'installation.
- 3. Cliquez sur **Installer**.
- 4. Ouvrez ensuite le launcher Minecraft.
- 5. Sélectionnez le profil OptiFine.

### Ajouter un shader

- 1. Lancez Minecraft avec OptiFine.
- 2. Ouvrez les **Options**.
- 3. Cliquez sur **Paramètres vidéo**.
- 4. Sélectionnez **Shaders**.
- 5. Cliquez sur **Dossier des shaders**.
- 6. Déposez votre fichier ZIP dans ce dossier.
- 7. Revenez dans Minecraft.
- 8. Sélectionnez le shader souhaité.

Le shader sera immédiatement appliqué.

## 🚀 Installer des shaders avec Iris Shaders

Iris est une alternative moderne à OptiFine, souvent utilisée avec Sodium pour obtenir de meilleures performances.

### Installer Iris

- 1. Téléchargez l'installateur Iris.
- 2. Lancez l'installation.
- 3. Activez Sodium si proposé.
- 4. Sélectionnez votre version de Minecraft.
- 5. Cliquez sur **Installer**.

### Ajouter un shader

- 1. Lancez Minecraft avec Iris.
- 2. Ouvrez les paramètres vidéo.
- 3. Accédez au menu **Shaders**.
- 4. Importez votre pack de shaders.
- 5. Activez-le dans la liste disponible.

## 🌅 Les shaders Minecraft les plus populaires

Parmi les packs les plus connus :

- BSL Shaders
- Complementary Shaders
- SEUS Renewed
- Sildur's Vibrant Shaders
- Continuum Shaders

Chaque shader propose un rendu différent, allant d'un style proche du jeu original à des graphismes extrêmement réalistes.

## ⚙️ Optimiser les performances

Les shaders peuvent fortement solliciter votre ordinateur.

Pour améliorer les performances :

- Réduisez la distance d'affichage.
- Désactivez certains effets avancés.
- Utilisez Sodium avec Iris.
- Limitez les packs de ressources très lourds.
- Testez plusieurs niveaux de qualité proposés par le shader.

## 🛠️ Résoudre les problèmes courants

### Mon shader ne s'affiche pas

Vérifiez que :

- Le fichier est bien au format ZIP.
- Le shader est compatible avec votre version de Minecraft.
- Vous utilisez OptiFine ou Iris.

### J'ai des chutes de FPS

Essayez de :

- Réduire les paramètres graphiques.
- Utiliser un shader moins gourmand.
- Installer Sodium avec Iris.
- Fermer les applications inutiles en arrière-plan.

### Minecraft plante au lancement

Assurez-vous :

- D'utiliser une version compatible du shader.
- D'avoir suffisamment de mémoire allouée à Minecraft.
- De mettre à jour vos pilotes graphiques.

## 💡 Conseils

- Testez plusieurs shaders afin de trouver celui qui correspond à votre configuration.
- Sauvegardez vos paramètres avant de changer de pack.
- Évitez les shaders très gourmands sur les ordinateurs anciens.
- Vérifiez régulièrement les mises à jour des shaders.
