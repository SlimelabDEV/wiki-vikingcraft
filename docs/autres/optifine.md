# ⚡ Installer OptiFine sur Minecraft

OptiFine est un outil très populaire pour Minecraft Java Edition. Il permet d'améliorer les performances, d'ajouter des options graphiques avancées et d'utiliser des shaders.

Dans ce guide, découvrez comment installer OptiFine et le configurer correctement.

## 📋 Prérequis

Avant de commencer, vous devez disposer :

- De Minecraft Java Edition
- Du launcher Minecraft officiel
- De Java installé sur votre ordinateur pour lancer l'installateur
- D'une version d'OptiFine compatible avec votre version de Minecraft

## 📥 Télécharger OptiFine

Téléchargez toujours une version d'OptiFine correspondant à votre version de Minecraft.

Avant l'installation, vérifiez :

- La version exacte de Minecraft
- La compatibilité d'OptiFine
- La présence éventuelle d'une version preview si votre version de Minecraft est récente

Une version incompatible peut empêcher Minecraft de démarrer.

## ⚙️ Installer OptiFine

### Méthode classique

- 1. Téléchargez le fichier `.jar` d'OptiFine.
- 2. Lancez le fichier.
- 3. Cliquez sur **Install**.
- 4. Attendez la confirmation d'installation.
- 5. Ouvrez le launcher Minecraft.

Un profil OptiFine apparaîtra automatiquement dans le launcher.

### Si le fichier ne s'ouvre pas

Assurez-vous que Java est installé.

Vous pouvez faire un clic droit sur le fichier OptiFine, puis sélectionner **Ouvrir avec Java**.

## 🚀 Lancer Minecraft avec OptiFine

Pour utiliser OptiFine :

- 1. Ouvrez le launcher Minecraft.
- 2. Sélectionnez le profil OptiFine.
- 3. Cliquez sur **Jouer**.
- 4. Ouvrez les options vidéo.

Vous devriez voir de nouveaux réglages graphiques disponibles.

## ✨ Utiliser des shaders avec OptiFine

OptiFine permet d'utiliser des packs de shaders.

Pour ajouter un shader :

- 1. Lancez Minecraft avec OptiFine.
- 2. Ouvrez **Options**.
- 3. Cliquez sur **Paramètres vidéo**.
- 4. Sélectionnez **Shaders**.
- 5. Cliquez sur **Dossier des shaders**.
- 6. Déposez le fichier ZIP du shader.
- 7. Activez-le dans la liste.

Le shader sera appliqué directement en jeu.

## ⚙️ Optimiser les performances avec OptiFine

Pour gagner des FPS, vous pouvez :

- Réduire la distance d'affichage.
- Désactiver les ombres dynamiques.
- Mettre les graphismes en rapide.
- Désactiver les nuages.
- Réduire les particules.
- Activer les options de performance proposées.

Les meilleurs réglages dépendent de votre ordinateur.

## 🛠️ Résoudre les problèmes courants

### OptiFine n'apparaît pas dans le launcher

Essayez de :

- Relancer le launcher Minecraft.
- Réinstaller OptiFine.
- Vérifier que Minecraft a déjà été lancé une fois dans la même version.

### Minecraft plante au lancement

Vérifiez que :

- La version d'OptiFine correspond à votre version de Minecraft.
- Aucun mod incompatible n'est installé.
- Java est correctement installé.

### Les shaders ne fonctionnent pas

Assurez-vous que :

- Le shader est au format ZIP.
- Le shader est compatible avec votre version.
- Votre ordinateur est assez performant.

## 💡 Conseils

- Testez vos réglages progressivement.
- Utilisez des shaders légers si votre ordinateur est ancien.
- Gardez une copie de vos paramètres si vous changez souvent de version.
- Évitez de combiner OptiFine avec des mods incompatibles.
