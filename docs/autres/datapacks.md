# 📦 Installer un datapack sur Minecraft

Les datapacks permettent d'ajouter ou modifier des fonctionnalités dans Minecraft Java Edition sans installer de mod. Ils peuvent changer les recettes, ajouter des structures, modifier le loot, créer des systèmes de progression ou ajouter des mécaniques personnalisées.

Dans ce guide, découvrez comment installer un datapack sur Minecraft.

## 📋 Prérequis

Avant de commencer, vous devez disposer :

- De Minecraft Java Edition
- D'un monde solo ou d'un serveur
- D'un datapack compatible avec votre version
- D'un accès au dossier du monde

## 📥 Télécharger un datapack Minecraft

Avant de télécharger un datapack, vérifiez :

- Sa version Minecraft compatible
- Les fonctionnalités ajoutées
- Les éventuelles commandes nécessaires
- Les instructions données par le créateur

Un datapack prévu pour une ancienne version peut ne pas fonctionner correctement sur une version récente.

## 💻 Installer un datapack en solo

### Depuis le menu Minecraft

- 1. Lancez Minecraft.
- 2. Cliquez sur **Solo**.
- 3. Sélectionnez votre monde.
- 4. Cliquez sur **Modifier**.
- 5. Cliquez sur **Ouvrir le dossier du monde**.
- 6. Ouvrez le dossier `datapacks`.
- 7. Déposez le fichier ZIP du datapack.
- 8. Lancez le monde.

Le datapack sera chargé automatiquement.

## 📂 Installer un datapack manuellement

Vous pouvez aussi accéder directement au dossier Minecraft.

Sous Windows :

```text
%appdata%\.minecraft\saves
```

Ouvrez le dossier de votre monde, puis le dossier :

```text
datapacks
```

Déposez le fichier ZIP du datapack dans ce dossier.

## 🔄 Activer un datapack

Une fois en jeu, vous pouvez utiliser la commande :

```text
/reload
```

Pour vérifier les datapacks actifs :

```text
/datapack list
```

Si le datapack apparaît dans la liste, il est bien chargé.

## 🌐 Installer un datapack sur un serveur Minecraft

Pour un serveur :

- 1. Arrêtez le serveur.
- 2. Ouvrez le dossier du monde principal.
- 3. Accédez au dossier `datapacks`.
- 4. Ajoutez le fichier ZIP du datapack.
- 5. Redémarrez le serveur.
- 6. Utilisez `/datapack list` pour vérifier.

Il est conseillé de faire une sauvegarde du monde avant l'installation.

## 🛠️ Résoudre les problèmes courants

### Le datapack ne fonctionne pas

Vérifiez que :

- Le fichier ZIP est dans le bon dossier.
- Le datapack est compatible avec votre version.
- Vous avez utilisé la commande `/reload`.
- Le datapack n'est pas dans un dossier supplémentaire.

### Minecraft indique une erreur

Le datapack peut être trop ancien ou mal structuré.
Consultez les instructions du créateur et testez avec la version recommandée.

### Les fonctionnalités n'apparaissent pas

Certains datapacks nécessitent :

- Une commande d'activation
- Une progression spécifique
- La génération de nouveaux chunks
- Un redémarrage du monde

## 💡 Conseils

- Sauvegardez votre monde avant d'ajouter un datapack.
- Installez les datapacks un par un.
- Vérifiez les commandes fournies par le créateur.
- Évitez de combiner trop de datapacks qui modifient les mêmes mécaniques.
