# 🧵 Installer Fabric sur Minecraft

Fabric est un chargeur de mods léger et moderne pour Minecraft Java Edition. Il est souvent utilisé pour les mods d'optimisation, les petits ajouts de gameplay, les interfaces améliorées et certains packs de mods récents.

Dans ce guide, découvrez comment installer Fabric sur Minecraft et ajouter vos premiers mods.

## 📋 Prérequis

Avant de commencer, vous devez disposer :

- De Minecraft Java Edition
- Du launcher Minecraft officiel
- D'une connexion Internet
- De l'installateur Fabric
- De mods compatibles avec Fabric

## 📥 Télécharger Fabric

Fabric doit être installé dans une version compatible avec votre version de Minecraft.

Avant d'installer un mod, vérifiez :

- La version de Minecraft demandée
- La version de Fabric nécessaire
- La présence éventuelle de **Fabric API**

De nombreux mods Fabric nécessitent le mod **Fabric API** pour fonctionner correctement.

## ⚙️ Installer Fabric

### Méthode rapide

- 1. Téléchargez l'installateur Fabric.
- 2. Lancez le fichier d'installation.
- 3. Choisissez votre version de Minecraft.
- 4. Sélectionnez le launcher Minecraft.
- 5. Cliquez sur **Installer**.

Une fois l'installation terminée, un profil Fabric sera ajouté dans le launcher.

## 📦 Installer Fabric API

Fabric API est une dépendance très fréquente pour les mods Fabric.

Pour l'installer :

- 1. Téléchargez Fabric API pour votre version de Minecraft.
- 2. Ouvrez le dossier `.minecraft`.
- 3. Placez le fichier `.jar` dans le dossier `mods`.
- 4. Lancez Minecraft avec le profil Fabric.

Sans Fabric API, certains mods peuvent ne pas se charger.

## 📂 Ajouter des mods Fabric

### Sous Windows

Appuyez sur `Windows + R`, puis entrez :

```text
%appdata%\.minecraft
```

Ouvrez le dossier `mods`, puis déposez vos fichiers `.jar`.

### Sous macOS

Le dossier Minecraft se trouve ici :

```text
~/Library/Application Support/minecraft
```

### Sous Linux

Le dossier Minecraft se trouve ici :

```text
~/.minecraft
```

Si le dossier `mods` n'existe pas, créez-le.

## 🚀 Lancer Minecraft avec Fabric

Pour utiliser vos mods :

- 1. Ouvrez le launcher Minecraft.
- 2. Sélectionnez le profil Fabric.
- 3. Cliquez sur **Jouer**.
- 4. Vérifiez que vos mods sont bien chargés.

Certains mods ajoutent un menu dédié, tandis que d'autres fonctionnent directement en arrière-plan.

## ⚡ Pourquoi utiliser Fabric ?

Fabric est apprécié pour :

- Sa légèreté
- Ses bonnes performances
- Sa compatibilité avec de nombreux mods d'optimisation
- Ses mises à jour rapides

Il est souvent utilisé avec des mods comme Sodium, Lithium, Mod Menu ou Iris Shaders.

## 🛠️ Résoudre les problèmes courants

### Minecraft ne démarre pas

Vérifiez que :

- Vos mods sont compatibles avec Fabric.
- Vous n'avez pas installé un mod prévu pour Forge.
- Fabric API est installé si nécessaire.
- La version de Minecraft correspond bien aux mods.

### Un mod n'apparaît pas

Assurez-vous que :

- Le fichier `.jar` est dans le dossier `mods`.
- Vous lancez le profil Fabric.
- Le mod n'a pas besoin d'une dépendance supplémentaire.

### Message d'erreur au lancement

Lisez le message affiché par Minecraft.
Il indique souvent le mod incompatible, la dépendance manquante ou la version incorrecte.

## 💡 Conseils

- Installez Fabric API dès le départ.
- Ne mélangez pas les mods Forge et Fabric.
- Gardez une copie de vos mondes avant d'ajouter des mods.
- Testez le jeu après chaque nouveau mod installé.
