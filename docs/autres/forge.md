# 🔧 Installer Forge sur Minecraft

Forge est l'un des chargeurs de mods les plus utilisés sur Minecraft Java Edition. Il permet d'installer des mods pour ajouter de nouveaux blocs, objets, créatures, interfaces, dimensions ou mécaniques de jeu.

Dans ce guide, découvrez comment installer Forge sur Minecraft et l'utiliser correctement avec vos mods.

## 📋 Prérequis

Avant de commencer, vous devez disposer :

- De Minecraft Java Edition installé
- Du launcher Minecraft officiel
- D'une connexion Internet
- De Java installé sur votre ordinateur pour lancer l'installateur
- D'un mod compatible avec Forge

## 📥 Télécharger Forge

Pour installer Forge, vous devez télécharger une version compatible avec votre version de Minecraft.

Avant le téléchargement, vérifiez :

- La version exacte de Minecraft utilisée
- La version de Forge demandée par vos mods
- La présence éventuelle de dépendances supplémentaires

Un mod prévu pour une version précise de Minecraft ne fonctionne généralement pas sur une autre version.

## ⚙️ Installer Forge

### Méthode classique

- 1. Téléchargez l'installateur Forge.
- 2. Lancez le fichier `.jar`.
- 3. Sélectionnez **Install client**.
- 4. Cliquez sur **OK**.
- 5. Attendez la fin de l'installation.

Une fois l'installation terminée, un nouveau profil Forge apparaîtra dans le launcher Minecraft.

### Si l'installateur ne s'ouvre pas

Vérifiez que Java est bien installé sur votre ordinateur.

Vous pouvez également faire un clic droit sur le fichier puis choisir **Ouvrir avec Java**.

## 📂 Ajouter des mods avec Forge

Après avoir installé Forge, vous devez placer vos mods dans le dossier Minecraft.

### Sous Windows

Appuyez sur `Windows + R`, puis entrez :

```text
%appdata%\.minecraft
```

Ouvrez ensuite le dossier `mods`.

### Sous macOS

Le dossier Minecraft se trouve ici :

```text
~/Library/Application Support/minecraft
```

### Sous Linux

Le dossier Minecraft se trouve généralement ici :

```text
~/.minecraft
```

Si le dossier `mods` n'existe pas, créez-le manuellement.

Déplacez ensuite les fichiers `.jar` de vos mods dans ce dossier.

## 🚀 Lancer Minecraft avec Forge

Pour utiliser Forge :

- 1. Ouvrez le launcher Minecraft.
- 2. Sélectionnez le profil Forge.
- 3. Cliquez sur **Jouer**.
- 4. Attendez le chargement du jeu.

Si Forge est correctement installé, un menu ou une indication liée aux mods apparaîtra dans Minecraft.

## 🛠️ Résoudre les problèmes courants

### Minecraft plante au lancement

Vérifiez que :

- Le mod est compatible avec votre version de Minecraft.
- Le mod est bien prévu pour Forge.
- Toutes les dépendances sont installées.
- Vous n'avez pas mélangé des mods Forge et Fabric.

### Forge n'apparaît pas dans le launcher

Essayez de :

- Relancer le launcher Minecraft.
- Réinstaller Forge.
- Vérifier que vous avez installé Forge en mode **client**.

### Le mod ne fonctionne pas

Assurez-vous que :

- Le fichier `.jar` est dans le dossier `mods`.
- Vous lancez le bon profil Forge.
- Le mod n'est pas compressé dans un fichier ZIP.

## 💡 Conseils

- Installez les mods un par un pour repérer facilement les erreurs.
- Sauvegardez vos mondes avant d'ajouter de nouveaux mods.
- Utilisez toujours des mods provenant de sources fiables.
- Gardez la même version de Minecraft, de Forge et des mods.
