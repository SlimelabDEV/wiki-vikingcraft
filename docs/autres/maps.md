# 🗺️ Installer une map sur Minecraft

Les maps Minecraft permettent de jouer sur des mondes créés par d'autres joueurs : aventures, mini-jeux, villes, bases, maps survie, maps parkour ou constructions impressionnantes.

Dans ce guide, découvrez comment installer une map sur Minecraft Java Edition et Bedrock Edition.

## 📋 Prérequis

Avant de commencer, vous devez disposer :

- D'une version de Minecraft installée
- D'une map compatible avec votre édition
- D'un fichier ZIP, dossier de monde ou fichier `.mcworld`
- D'un accès aux fichiers du jeu

## 📥 Télécharger une map Minecraft

Avant de télécharger une map, vérifiez :

- La version de Minecraft recommandée
- L'édition compatible, Java ou Bedrock
- Les éventuels packs de ressources nécessaires
- Les règles indiquées par le créateur

Une map ancienne peut fonctionner, mais certains blocs ou systèmes peuvent être cassés sur les versions récentes.

## 💻 Installer une map sur Minecraft Java Edition

### Étapes d'installation

- 1. Téléchargez la map.
- 2. Décompressez le fichier ZIP si nécessaire.
- 3. Ouvrez le dossier `.minecraft`.
- 4. Accédez au dossier `saves`.
- 5. Déplacez le dossier de la map dans `saves`.
- 6. Lancez Minecraft.
- 7. Ouvrez le menu **Solo**.
- 8. Sélectionnez la map dans la liste.

### Dossier Minecraft sous Windows

Appuyez sur `Windows + R`, puis entrez :

```text
%appdata%\.minecraft
```

Le dossier des mondes se trouve ici :

```text
.minecraft\saves
```

## 📱 Installer une map sur Minecraft Bedrock Edition

### Avec un fichier `.mcworld`

- 1. Téléchargez la map au format `.mcworld`.
- 2. Ouvrez le fichier.
- 3. Minecraft se lance automatiquement.
- 4. Attendez l'importation du monde.
- 5. Lancez la map depuis la liste des mondes.

### Sur Android

Vous pouvez ouvrir le fichier `.mcworld` depuis votre gestionnaire de fichiers.
Minecraft importera automatiquement la map.

### Sur iPhone et iPad

Téléchargez le fichier dans l'application Fichiers, puis ouvrez-le avec Minecraft.

## 📦 Installer un pack de ressources fourni avec la map

Certaines maps utilisent un pack de ressources spécifique.

Si un pack est fourni :

- Lisez le fichier d'instructions fourni avec la map.
- Installez le pack dans le dossier `resourcepacks`.
- Activez-le avant de lancer la map.

Cela permet d'afficher correctement les textures, sons ou interfaces personnalisées.

## 🛠️ Résoudre les problèmes courants

### La map n'apparaît pas

Vérifiez que :

- Le dossier de la map est bien dans `saves`.
- Le fichier ZIP a été décompressé.
- Le dossier contient bien les fichiers du monde.

### La map se lance mais semble vide

Vous avez peut-être placé le mauvais dossier dans `saves`.
Ouvrez le dossier de la map et vérifiez qu'il contient un fichier comme `level.dat`.

### Certains blocs sont manquants

Essayez de :

- Lancer la map avec la version recommandée.
- Installer le pack de ressources fourni.
- Vérifier si la map nécessite des mods.

## 💡 Conseils

- Sauvegardez vos mondes avant de manipuler les fichiers.
- Lisez toujours les instructions fournies avec la map.
- Utilisez la version recommandée par le créateur.
- Évitez les maps provenant de sources suspectes.
